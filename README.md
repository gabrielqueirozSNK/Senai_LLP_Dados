# Simulador de Dados (Senai_LLP_Dados)
Este projeto é uma aplicação de terminal que simula o lançamento de dados. Ele gera números aleatórios e exibe o resultado visualmente na tela.
## Funcionalidades
O programa realiza as seguintes operações:
- Pergunta ao usuário a quantidade de dados a serem lançados.
- Caso o usuário não informe um valor, o sistema considera o lançamento de 1 dado por padrão.
- Gera um número aleatório entre 1 e 6 para cada dado, utilizando a função Math.random().
- Retorna no terminal o desenho da face do dado correspondente ao valor que foi sorteado.
